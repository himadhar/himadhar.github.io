JavaScript (JS)

Here, I am planning to cover areas in JavaScript(JS) that helps in brushing up one's skills on various avenues of programming in JS. We know that JS is a loosely typed language. This means, many of the things we know from other paradigms of programming differs. For example, 1 == "1" in Java returns false, whereas the same returns true in JS. Below are a series of questions focussed on basics on JS targeting various basics to keep in mind while programming (specifically) in JS. Feel free to give it a try and see where you stand. Read about the questions that you need help in understanding. The aim of this post is to help you in honing/brushing/re-collecting your knowledge on JS.
